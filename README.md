# Temple React App

A minimal React app scaffolded with Vite. This project requests Node 22 via `.nvmrc` and `package.json` engines.

Quick start

1. Install Node 22 (nvm example):

```bash
nvm install 22
nvm use 22
```

Or with Volta:

```bash
volta install node@22
```

2. Install dependencies and start dev server:

```bash
npm install
npm run dev
```

Build:

```bash
npm run build
npm run preview
```
