# Public assets

Drop static assets here that should be served as-is.

To override the hero image used on the homepage, add a file at:

```
public/assets/hero.jpg
```

Event thumbnails: to replace Unsplash event images with local files, drop images as:

```
public/assets/events/event-1.jpg
public/assets/events/event-2.jpg
...event-9.jpg
```

If a local event image exists the browser will use it; otherwise the app falls back to a royalty-free Unsplash image.
